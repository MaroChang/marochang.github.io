### Write the script

```sh:n
#!/bin/bash

#A simple script
name="CDC"
echo "Hello, $name"

#List all of command
history

#Pause the terminal
read -p "press any key to exit ..."
```

- `#!` Shebang: Always start the script with `#!/bin/bash` to specify the interpreter

#### Make the Script executable

change mod
```sh
chmod +x script.sh
```

#### Run the Script

```sh
./script.sh
```

```sh
bash script.sh
```
OR
```sh
source script.sh
```
